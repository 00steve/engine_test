§ Draw()

void View::Draw ( )
virtual

Where the node is able to draw itself.

If using the base implementation, the node will just render all of it's children nodes. If implementing, which I assume most nodes will do, you can just call Node::Draw() to have it draw all of the child nodes, otherwise it is up to you to make sure children are rendered.

Reimplemented from Node.
