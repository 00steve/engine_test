§ DefaultEnabled() [1/2]

bool Physical::DefaultEnabled ( )
virtual

this is a generic function that returns if the physical object is enabled or not.

If it is, things should collide against it and its bodies should still be active in the simulation. If this is set to false, none of the bodies that constitutes the physical should be enabled.

§ SurfaceParameters()

dSurfaceParameters * Physical::SurfaceParameters ( dGeomID  )
virtual

return the contact settings of a particular geom that is part of this physical object.

By default nothing is returned. This can be overridden so that it returns any settings based on the geom id that is passed to it. This allows the programmer to let physicals return a different contact_setting for each contact part of it.
