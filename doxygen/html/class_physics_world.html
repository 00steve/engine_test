static

Creates new plane in the current physics space and world.

The normal of the normal is used to determine the angle of the plane, the offset is used to determine how far away from the origin the plane sits.

Since this plane does not have a body, it will be static by default, meaning that it has no mass and cannot be moved by any objects. It is an immovable object.
